1. In jQuery terminology, what is the difference between the “jQuery function” and the “jQuery object”?

The jQuery function is the value of jQuery or of $. This is the function that creates jQuery objects and register handlers to be invoked when the DOM is ready. It also serves jquery as namespace. A JQuery object is an object returned by the jQuery function.
